## 🛠️ &nbsp; Getting Started

</div>

### Prerequisites

- **Node.js** v18 or later → [Download](https://nodejs.org/)
- **npm** (bundled with Node.js)
- **Git** → [Download](https://git-scm.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/mohitsharmaa21/mohit-portfolio.git

# 2. Navigate into the project
cd mohit-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open **[http://localhost:5173](http://localhost:5173)** in your browser. 🎉

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder, ready for deployment.

## 🚀 &nbsp; Deployment

</div>

<details>
<summary><b>▶ Deploy on Vercel (Recommended)</b></summary>
<br/>

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import this repository
4. Vercel auto-detects Vite — click **Deploy**
5. Your portfolio goes live at `https://your-project.vercel.app` ✅

</details>

<details>
<summary><b>▶ Deploy on Netlify</b></summary>
<br/>

1. Go to [netlify.com](https://netlify.com) → **Add New Site**
2. Connect your GitHub repository
3. Build settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
4. Click **Deploy Site** ✅

</details>
